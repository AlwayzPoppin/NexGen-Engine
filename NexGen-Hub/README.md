## ğŸ› ï¸ Getting Started

### Prerequisites
- Node.js (Latest LTS)
- Gemini API Key ([Get one here](https://aistudio.google.com/app/apikey))

### Installation
1.  **Clone & Install**:
    ```bash
    git clone https://github.com/AlwayzPoppin/NexGen-Hub.git
    cd NexGen-Hub
    npm install
    ```
2.  **Environment Setup**:
    - Create a `.env.local` file in the root directory.
    - Add: `VITE_API_KEY=your_gemini_api_key_here`
3.  **Ignite the Hub**:
    ```bash
    npm run dev
    ```
